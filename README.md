📚 Bookstore Analytics & SQL Lab
Аналитическая платформа для книжного магазина, демонстрирующая полный цикл работы с данными: от архитектуры БД до бизнес-инсайтов.
✨ Основные разделы
📊 Dashboard: Интерактивная визуализация KPI (выручка, средний чек, топ продаж, когортный анализ удержания).
💻 SQL Lab: Библиотека из 30+ сложных запросов (RFM-сегментация, LTV, ABC-анализ, скользящие средние).
🏗 Data Schema: Визуализация архитектуры базы данных по типу «Звезда» (Star Schema).
🛠 Стек технологий
Frontend: React 19, TypeScript, Vite.
Графики: Recharts.
Стили: Tailwind CSS.
Анимации: Motion (Framer Motion).
